**1.用户登录**
----------
  * 访问路径：<http://**.**.**.**:8080/user/login>
  * 调用方式：POST
  * 接口说明：用户登录
  * 接口参数：

| 字段        | 类型 |说明
| --- |:---:| ---:
|user|String|导入用户信息

  
* 接口返回：

|字段|说明
|-|-
|message|操作成功返回success，失败返回具体错误（与异常说明的描述相同）
|code|成功返回200，失败返回具体错误码



登录成功示例：
```java
{
  "msg": "success",
  "code": "200"
}
```

登录失败示例：
```java
{
  "msg": "用户名或密码错误",
  "code": "100102"
}
```


 * 返回错误码：

| subCode   | 描述 |
| --------   | -----  |
|100101|不存在该用户!|
|100102|用户名或密码错误!|
